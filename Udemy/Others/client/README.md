# Client Project Setup

- Installed packages of `client/`:

```cmd
cd client/
npm create vite@latest
```

- Press `.`
- Select `React` & `JavaScript + SWC`

```cmd
npm i
npm install @mui/material @emotion/react @emotion/styled
npm i socket.io-client
```

- Run the client

```cmd
npm run dev
```
